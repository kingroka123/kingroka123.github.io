Install Macro


Windows

Run the installer and log in to Macro. The Macro desktop app must be running on a computer. However, the remote and client devices do not have to be connected to the same internet connection. Consider adding macro to the start-up programs.


Other Devices (Chrome Browser)

Android/IOS: Add this page to your homescreen. Click on the newly created icon to access the app


All devices running Chrome (and possibly other browsers) can use Macro in the browser. Just click login to begin.
